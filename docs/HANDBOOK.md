# 标注员手册

本手册说明每个标签的含义、判断方法和常见例子。建议第一次标注前通读一遍。

---

## 标注流程

每个任务是一句话（一个人的一段发言）。

1. **点击任务** → 进入标注界面
2. **看上文（蓝色框）** — 了解这段话之前说了什么，帮助理解语境
3. **点击 ► 播放音频** — 完整听完
4. **修正转写（如有错误）** — 如果 ASR 写错了，可以直接在文本框里改
5. **选情感** — 根据听到的声音判断，不是只看文字
6. **选销售策略** — 这段话是销售说的，在做什么？（客户说的选 N/A）
7. **选客户意图** — 这段话是客户说的，什么态度？（销售说的选 N/A）
8. **勾副语言** — 听到什么勾什么，可以多选
9. **点 Submit** — 提交，进入下一条

## 情感标注

**最重要原则：听声音，不只看文字。**
同一句话，语气不同，情感完全不同。

| 标签 | 含义 | 典型表现 |
|------|------|---------|
| **0-生气** | 愤怒、被打扰、语气强硬 | 语速快、音调高、说话冲，如"你别打了！" |
| **1-厌恶** | 反感、厌烦、不耐烦 | 语气敷衍、叹气后接短回应，如"行行行知道了" |
| **2-恐惧** | 担心被骗、不安 | 声音变小、犹豫，如"这个……是不是骗局啊" |
| **3-高兴** | 感兴趣、开心、认可 | 语气轻快、有笑声，如"哦这样啊，挺好的" |
| **4-中立** | 正常对话，无明显情绪 | 平稳语气，大部分句子是这个 |
| **5-其他** | 敷衍、客套，或其他难以归类 | "嗯嗯嗯"、"哦好的"（语气很淡） |
| **6-伤心** | 失望、沮丧 | 说话低沉，如"唉，这个月又白忙了" |
| **7-惊讶** | 意外、不敢相信 | 语气上扬，如"啊？真的假的？" |
| **8-未知** | 音质差无法判断 | 背景噪声严重、声音模糊 |

**难点：**
- 短回应"嗯""好"很难判断 → 听语气，轻快=中立，拉长=厌恶，短促=中立
- 礼貌性的"谢谢""好的" → 通常是中立或其他，不是高兴

## 销售策略（客户的话选 N/A-客户turn）

| 标签 | 含义 | 例子 |
|------|------|------|
| **开场白/自我介绍** | 说明自己是谁、来自哪里 | "您好，我是伽马账的小张" |
| **需求挖掘** | 问客户的情况、痛点 | "您现在是自己记账还是找人做的？" |
| **产品介绍** | 介绍产品功能、优势 | "我们这个小程序可以一键报税" |
| **异议处理** | 解答客户的疑虑和拒绝 | "您说太贵了，但是我们比市面上便宜30%" |
| **逼单/促成** | 推动客户下决定 | "您看今天能不能定下来？" |
| **价格谈判** | 谈价格、优惠 | "给您申请一下折扣，三个月只要XX元" |
| **关系维护/寒暄** | 拉家常、问候 | "最近生意怎么样？" |
| **收尾/挂断** | 结束通话 | "好的，那您考虑一下，我改天再联系您" |

## 客户意图（销售的话选 N/A-销售turn）

| 标签 | 含义 | 例子 |
|------|------|------|
| **有兴趣/积极回应** | 感兴趣、愿意了解 | "哦这样啊，那你说说看" |
| **中性/观望** | 不置可否，听着 | "嗯""好""是吗" |
| **软拒绝（找借口）** | 有借口，不是直接拒绝 | "我最近比较忙，过段时间再说" |
| **硬拒绝（直接说不）** | 直接表示不需要 | "不需要，不用了，别打了" |
| **提问/深入了解** | 主动追问细节 | "多少钱一年？包括什么服务？" |
| **讨价还价** | 谈价格 | "能不能便宜点？" |
| **无关回应/跑题** | 答非所问，或聊别的 | "我刚才在开会，你说什么？" |

## 副语言（多选，听到什么勾什么）

这些是声音里的非言语信息，要用耳朵听，不是看文字。

| 标签 | 含义 |
|------|------|
| **笑声** | 明显听到笑声或含笑的语气 |
| **叹气** | 叹气声（"唉——"） |
| **犹豫/吞吐** | 明显停顿、卡壳、"那个……这个……" |
| **填充词** | 频繁出现"就是""然后""那个"等 |
| **语气词** | 语句末尾的"嘛""呀""哦""嘛" |
| **清嗓** | 说话前清嗓子 |
| **拖音** | 某个字拉得很长，如"好——" |
| **惊声** | 类似"哎"这种表达惊讶的声音 |
| **无** | 以上都没有，选这个 |

**注意：** 不能同时选"无"和其他选项。

## 常见问题

**Q：这段话很短，只有"嗯"，怎么判断情感？**
A：听语气。轻快短促 → 中立；拉长无奈 → 厌恶；愉快 → 高兴。

**Q：这段话又有销售说话又有客户回应（Diarization 没分开），怎么标？**
A：按主要说话人来标，在备注里写"含客户回应"。

**Q：音质很差听不清，怎么办？**
A：情感选 8-未知，其他标签按文字内容猜，在备注里写"音质差"。

**Q：我对标签不确定，怎么办？**
A：选你最直觉的那个，在备注里写"不确定"。不要空着不选。
